## Title: AI Skill Assessment & Learning Plan Agent
## Overview

This project is an AI-based system that analyzes a candidate’s resume and job description, evaluates their actual skill proficiency level using AI generated questions, and provide a personalized learning plan.

## Features

* Skill difference analysis like matched, missing and fing weak skils with interacted questioning Assignment.
* Skill evaluation with levels (Beginner / Intermediate / Strong)
* Personalized learning plan to improve weak skills

## Tech Stack

* Frontend: React (Vite)
* Backend: Node.js, Express
* AI: Google Gemini API

## How It Works

User enters a job description and resume then navigate to next page using Analyse button
 after System analyzes skills then Generates questions and take 
User answers.
System evaluates answers and 
Provides learning plan.

## Setup

### Backend

cd backend
npm install
node index.js


### Frontend


cd frontend
npm install
npm run dev


## API Endpoints

* `/analyze`
* `/questions`
* `/evaluate`
* `/plan`
